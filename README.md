# Diwali Sales Analysis

## Project Overview

This project focuses on Exploratory Data Analysis (EDA) of Diwali Sales Data using Python. The objective was to clean, transform, and analyze customer purchase data to identify sales patterns, customer behavior, and business opportunities.

## Business Objective

To understand customer purchasing behavior during the Diwali season and identify the most profitable customer segments, states, occupations, and product categories.

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Data Cleaning

- Removed irrelevant columns
- Handled missing values
- Converted data types
- Checked for duplicate and null records

## Analysis Performed

### Customer Analysis
- Gender-wise purchasing behavior
- Age group analysis
- Marital status analysis
- Occupation analysis

### Geographic Analysis
- Top performing states
- Regional sales contribution

### Product Analysis
- Product category performance
- Top-selling products

## Key Insights

- Female customers contributed significantly to total sales.
- Customers aged 26–35 represented the highest purchasing segment.
- Uttar Pradesh, Maharashtra, and Karnataka generated the highest revenue.
- Married customers showed higher purchasing activity.
- Food, Clothing, and Electronics were among the top-performing categories.

## Business Recommendations

- Focus marketing campaigns on high-performing states.
- Target customers aged 26–35 with personalized offers.
- Increase inventory for top-selling product categories.
- Develop customer retention strategies for repeat buyers.
